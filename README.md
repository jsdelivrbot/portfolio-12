# To install:

1.  Clone Repo

2.  cd portfolio/

3.  run `(sudo) npm install`

4.  Start application
    ⋅⋅⋅a. **`npm start`** = `browser-refresh server.js`
    ⋅⋅⋅b. `npm run deploy` = `node server.js`
    ⋅⋅⋅c. `npm run nodemon` = `nodemon server.js`
